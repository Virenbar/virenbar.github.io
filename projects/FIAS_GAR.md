Проект БД ФИАС ГАР для SQL Server и приложение для её обновления.

* FIAS_GAR  
Проект БД для SQL Server.  
* FIAS.Core  
Библиотека для работы с БД FIAS_GAR и API ФИАС.
* FIASUpdate  
Приложение для импорта данный из XML в БД.
